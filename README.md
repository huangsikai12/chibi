# 上海商学院第二课堂抢课脚本使用文档
## 快捷使用方式
找到Release目录
下载对应系统的运行程序即可
## 一、环境
python3.12
## 二、依赖
安装requests依赖即可
## 三、JSESSION获取方式
进入二课后按一下键盘上的F12
找到Application并点击
之后找到Cookies展开
在右边找到JESSIONID后面的值复制那一串

## 四、活动ID获取方式
点开活动页面后在地址栏里复制hdfbsqid=后面的值一直到&(不包含&)
